# Hardware procurement — рекомендації для громади

> Цей документ — частина пілотного пакету TrashFlow. Він допомагає
> керівництву Прилуцької громади ухвалити рішення «що саме купляти», щоб
> софт почав економити пальне і час диспетчера. Ціни на залізо НЕ входять
> у вартість розробки софту і покриваються бюджетом громади / донорським
> фінансуванням.

## TL;DR — що треба купити

| Що | Кількість | Ціна за од. | Разом |
|---|---:|---:|---:|
| Ультразвукові датчики наповненості (публічні баки) | 200 | €100 | **€20 000** |
| QR-стікери (приватний сектор) | 4 000 | 5 ₴ (≈ €0.12) | **€480** |
| GPS-трекери на машинах | вже є | — | — |
| Розгортання + калібрування датчиків (1 раз) | — | — | **€3 000** |
| **Разом одноразово** | | | **≈ €23 500** |
| SIM-карти / NB-IoT тариф (на рік) | 200 | €5/рік | €1 000 |

**Окупність:** при −20% економії пального (консервативно; Madrid та Nitra
показали −30%) — **14–18 місяців**.

## 1. Датчики наповненості: ультразвук, не камери

### Чому ультразвук — стандарт індустрії

| Параметр | Ультразвук | Камера + CV |
|---|---|---|
| Ціна за бак | €50–150 | €300–800 |
| Енергія | 5+ років від батареї | щоденне заряджання / провід |
| Стійкість до зими/вологи | висока | падає (запотівання, обмерзання) |
| Що міряє | %% наповненості | %% + тип сміття (overkill для нашої задачі) |
| Хто стоїть на міській інфраструктурі | Мадрид (11 100), Прага (3 294), Амстердам, Барселона | експериментальні пілоти |
| Складність інтеграції | HTTP webhook кожні 4-6 год | відеопотік + GPU |

**Рішення:** ультразвук. Камеру для класифікації типу сміття ми вже маємо
на стороні резидента (фото в застосунку → YOLOv8 → категорія). Дублювати
це обладнанням на баку — даремна витрата 200 × €500 = €100 000.

### Конкретні моделі — від найкращої до найдешевшої

1. **Sensoneo Quatro** (€120/бак, NB-IoT)
   - Готовий webhook на наш API.
   - 5 років батареї, заводська гарантія 3 роки.
   - Підтримує підземні контейнери до 4 м.
   - Використовується в Прилуках Sensoneo через мерію Праги — є кейси
     надійної експлуатації за -25 °C.

2. **EMZ EcoMonitor** (€80/бак, LoRaWAN)
   - Якщо громада планує розгорнути LoRa-мережу для інших IoT — спільна
     інфраструктура.
   - Без NB-IoT абонплати.
   - Менший radius, потребує гетьвей кожні 2–5 км.

3. **Open-source: ESP32 + HC-SR04 + SIM7080** (~€25 матеріалів/бак)
   - Для пілота на 10–20 баків щоб довести концепцію.
   - Ремонт силами громади, без вендор-локу.
   - **Мінус:** немає сертифікації IP67, є ризик відмов узимку.

**Рекомендація:** змішати. 20 баків — open-source пілот за свої кошти
(перші 30 днів). Якщо ROI підтвердиться — закупити Sensoneo на решту.

## 2. GPS-трекери на машинах

Громада повідомила, що трекери вже стоять. Питання — який вендор + чи є
доступ до API.

Найпопулярніші в Україні:

| Вендор | API | Українська підтримка |
|---|---|---|
| **Wialon (Gurtam)** | Так, REST + SDK | Дуже добра |
| **Bitrek (Bitrek Connect)** | Так, REST | Добра |
| **Teltonika** | Через брокер (TAVL/Wialon) | Через інтегратора |
| **GPSWOX** | Так, REST | Базова |

**Дія:** дізнатися вендора, попросити в КП API ключ. Інтеграція з нашою
БД — 1 робочий день після отримання доступу. Адаптер уже написаний:
`services/gps/GpsProvider.ts`.

## 3. QR-стікери для приватного сектору

**ПВХ-стікер 50×50 мм з ламінацією**, цінник у місцевих типографіях
~5 ₴/штука. На 4 000 домогосподарств — 20 000 ₴ (≈ €450). Друк через
будь-яку місцеву типографію за 1 тиждень.

QR кодує `https://trashflow.app/my-bin/<token>`. Токен унікальний на бак.

### План розгортання

- **Тиждень 1:** друк 4 000 стікерів, розклеїти всім домогосподарствам
  (можна через ОСББ + старост вулиць).
- **Тиждень 2–3:** інформаційна кампанія (ФБ-група громади, листівки в
  пошту, оголошення на дверях).
- **Місяць 1–6:** збір QR-сканів. Тариф для всіх однаковий (`×1.0`),
  додаток дає бонус 50% (`×0.5`) за сортування. Це Phase 1 PAYT.
- **Рік +1:** запровадження Phase 2 PAYT — ті, хто жодного разу не
  сканував — переходять на `×1.5`. Юридично оформляється рішенням сесії
  міської ради про тариф на вивіз ТПВ.

## 4. Окупність — детальний розрахунок

Припущення (за даними Madrid + Nitra + Treviso):
- −20% пальних витрат (консервативно; реальні кейси показують −30 до −63%)
- Парк Прилук: 8 машин × 35 км/маршрут × 36 маршрутів/тиждень = 10 080 км/тиж
- Витрата 28 л/100км × 58 ₴/л = 16 ₴/км
- Поточні річні витрати на пальне: 10 080 × 52 × 16 ≈ 8.4 млн ₴
- Економія −20%: ≈ 1.68 млн ₴/рік (≈ €38 000/рік)

**Окупність обладнання + софту:**

```
Одноразові інвестиції:  €23 500
Річна економія:         €38 000
Окупність:              ≈ 7 місяців (датчики) / 14 міс. з урахуванням нашого пілот-софту
```

Це не враховує:
- Економія часу диспетчера (ще +€5 000–8 000/рік).
- Збільшення % сортування → менше штрафів від обласної СЕС.
- Можливість продати рішення сусіднім громадам як SaaS.
